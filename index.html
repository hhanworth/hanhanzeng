Zmotion多轴运动平台

课题组使用了Zmotion的运动控制板卡，配合驱动器实现对多轴（3轴）运动平台的控制。厂家提供的GUI应用程序只供测试使用，无法满足自动化采集要求； 而厂家实际上有提供 C++ 的SDK，通过一些 DLL 封装可以完美实现通过 python 对板卡的驱动。仓库为整理的板卡控制资料，以及大致接线方式。

由于该 Dll 只支持 win64 ，我的平台是 macOS ，选择使用 windows11 虚拟机运行 dll 驱动与macOS通过共享文件方式通讯实现整体控制（类似 Linux 共享内存方式）。

另外仓库中包含摄像头采集、深度学习训练等代码。
